# 💸 Assistente Financeiro IA

Protótipo desenvolvido para a disciplina de Inteligência Artificial. Este projeto consiste em um assistente financeiro inteligente em formato de chat, capaz de interpretar linguagem natural, extrair dados financeiros, classificar categorias e gerenciar um fluxo de caixa completo.

## 🚀 Funcionalidades
- **Chat Conversacional:** Informe ganhos e gastos conversando naturalmente (ex: "gastei 30 de uber e recebi 150 de um freela").
- **Agentes Autônomos:** A IA decide autonomamente quando registrar uma transação ou consultar o saldo.
- **Dashboard Dinâmico:** Painel com filtros em tempo real para acompanhar Receitas e Despesas.
- **Multi-sessão:** Memória de chat e banco de dados isolados por ID de usuário.

## 🛠️ Tecnologias Utilizadas
- **Interface:** Streamlit
- **Orquestração de IA:** Agno (Function Calling e Memória)
- **Modelo LLM:** Google Gemini 2.5 Flash
- **Banco de Dados:** SQLite3
- **Manipulação de Dados:** Pandas
